About Me

Developer & Designer

I am a web developer who is currently doing freelance work and is a student at Carleton University. I help maintain and create custom websites and components to help get businesses started and grow.

Front-end development is my strong suit but I have experience working in a full-stack environment. I regularly work with HTML/CSS and JavaScript and I have just started working with React and TypeScript as well as Amazon Web Services. Recently, I started working with React and TypeScript to built components for upcoming websites.

I have an intense drive to learn new things and grow as a developer. Currently, I am working on improving my UX/UI skills, learning about data structures and algorithims, and writing more server side code. When I'm not coding, I enjoy playing sports, video games, going to the gym, and hanging out with friends.
